# Russian_Text_Loc_Serious_Sam_2_v2.159
Русификатор текста для Serious Sam 2 без русской озвучки.

Инструкция по установке русификатора для Serious Sam 2

1. Извлечь содержимое архива в папку с Serious Sam 2

Путь папки с игрой по умолчанию:
C:\Program Files (x86)\Steam\steamapps\common\Serious Sam 2

2. Или заходим в библиотеку стима и нажимаем правой кнопкой мыши на игру Serious Sam 2 в списке выбираем:

Свойства->Установленные файлы->Обзор

Откроется папка с файлами игры, туда извлекаем все файлы русификатора.

3. Запустить игру / играть.

Приятной игры!
